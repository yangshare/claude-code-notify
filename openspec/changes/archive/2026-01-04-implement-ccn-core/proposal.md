# 变更：实现 Claude Code Notify (CCN) 核心功能

## 为什么

Claude Code 用户在执行长时间任务（如构建、测试、部署）时，需要频繁切换窗口查看执行状态，影响工作效率。现有的 CLI 输出方式不够直观，且缺乏视觉和听觉提醒。我们需要打造一个优雅的通知系统，让用户能够在任务完成时及时收到系统级通知。

## 变更内容

- **新增 CLI 核心功能**：实现无界面、轻量级的命令行工具 `ccn.exe`，支持 `notify`、`init`、`setup` 等命令
- **新增 Windows 原生通知**：实现符合 Windows 10/11 Fluent Design 规范的原生通知卡片
- **新增交互式配置向导**：提供现代化的终端交互界面 (TUI)，引导用户完成配置
- **新增自动集成功能**：实现 `ccn setup` 命令自动配置 Claude Code hooks
- **新增智能通知策略**：实现防打扰阈值、摘要聚合和场景化配置

## 影响

- **受影响规范**：所有新增功能（CLI 核心、通知显示、配置管理、自动集成、智能策略）
- **受影响代码**：这是一个全新的项目，所有代码将从零开始构建
- **技术栈选择**：需要选择合适的 Windows 通知库、CLI 框架和配置管理方案
